School Template — Cameroon Real Images Version
==============================================

This is the updated school template with real Cameroon school / education images included as local assets.

Files:
- index.html: placeholder-ready template for CMS/backend use
- demo.html: preview page using dummy school data
- styles.css: full responsive school design and animation CSS
- assets/: real Cameroon image files from Wikimedia Commons
- IMAGE_CREDITS.txt: attribution and license details for the included images
- placeholder-map.json: placeholder reference

Important:
- The HTML placeholders were kept intact in index.html.
- The demo page is only for preview.
- The images come from Wikimedia Commons and require attribution. Keep IMAGE_CREDITS.txt with the project, and review the licenses before using the images for a client website.
- Best final practice: replace demo photos with the actual school's own campus/classroom/gallery photos.
